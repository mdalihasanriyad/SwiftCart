# SwiftCart


১) null এবং undefined এর মধ্যে পার্থক্য কী?
undefined: জাভাস্ক্রিপ্ট নিজে থেকেই কোনো খালি ভেরিয়েবলকে এই মান দেয়।
null: এটি আমরা (ডেভেলপাররা) ইচ্ছা করে ব্যবহার করি কোনো ভেরিয়েবলকে "খালি" বা "শূন্য" বোঝানোর জন্য।

২) map() এবং forEach() এর মধ্যে পার্থক্য কী?
map(): এটি একটি নতুন অ্যারে তৈরি করে ফেরত দেয় (Return করে)।
forEach(): এটি কোনো কিছু ফেরত দেয় না। শুধুমাত্র লিস্টের প্রতিটি আইটেম নিয়ে কোনো কাজ করার জন্য এটি ব্যবহৃত হয়।

৩) == এবং === এর মধ্যে পার্থক্য কী?
== এটি শুধুমাত্র মান (Value) চেক করে। যেমন: 5 == "5" দিলে এটি true বলবে।
=== এটি মান এবং ডাটা টাইপ দুইটাই চেক করে। যেমন: 5 === "5" দিলে এটি false বলবে (কারণ একটি সংখ্যা, অন্যটি স্ট্রিং)।

৪) API ফেচ করার ক্ষেত্রে async/await কেন প্রয়োজন?
সার্ভার থেকে ডাটা আসতে কিছুটা সময় লাগে। async/await জাভাস্ক্রিপ্টকে বলে, "ডাটা না আসা পর্যন্ত অপেক্ষা করো"।

৫) JavaScript Scope (Global, Function, Block) কী?
Global: কোডের যেকোনো জায়গা থেকে এই ভেরিয়েবল ব্যবহার করা যায়।
Function: ভেরিয়েবলটি শুধুমাত্র ওই ফাংশনের ভেতরেই কাজ করবে।
Block: { } ব্র্যাকেটের (যেমন if বা for loop) ভেতরে ডিক্লেয়ার করা ভেরিয়েবল যা ওই ব্লকের বাইরে কাজ করে না।
